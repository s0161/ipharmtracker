🔧 Key Patterns

useSupabase Hook

Custom hook for auto-syncing React state to Supabase with camelCase/snake_case conversion

Task Engine

Template-driven task generation with role filtering and category grouping

Rotation Manager

Deterministic daily staff rotation for cleaning and task assignment

Compliance Scoring

Real-time score calculation across 4 pillars with trend history

Seed System

Versioned demo data (v29) with automatic orphaned key cleanup

Audit Logging

Automatic action tracking on all data modifications via logAudit()
